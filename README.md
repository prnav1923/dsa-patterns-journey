# 🧠 Patterns Covered

The problems are organized based on common **algorithmic patterns used in interviews**.

| Pattern               | Description                                   |
| --------------------- | --------------------------------------------- |
| Arrays & Hashing      | Fast lookups using hash maps and sets         |
| Two Pointers          | Efficient traversal using left/right pointers |
| Sliding Window        | Optimized subarray/substring problems         |
| Stack                 | Monotonic stack and parsing problems          |
| Binary Search         | Search optimization in sorted structures      |
| Linked List           | Pointer manipulation problems                 |
| Trees                 | Binary tree traversals and recursion          |
| Heap / Priority Queue | Top-K and streaming problems                  |
| Graphs                | BFS, DFS, and connectivity problems           |
| Backtracking          | Recursive exploration problems                |
| Dynamic Programming   | Optimization with overlapping subproblems     |

# 📝 Problem Template

Each solution file follows this structure:

```
Problem: Two Sum
Pattern: Arrays & Hashing

Approach:
Use a hashmap to store previously visited numbers and their indices.
For each number, check if (target - current) exists in the hashmap.

Time Complexity: O(n)

Space Complexity: O(n)
```
